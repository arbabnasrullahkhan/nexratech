<h2><img src="https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/solid/globe.svg" width="24" /> Platform Overview</h2>
<p>
  <b>Nexra PK</b> is an ultra-scalable, domain-independent Digital Ecosystem. Abandoning the traditional, bloated SPA (React/Vue) frameworks, Nexra is engineered using a custom <b>Vanilla JS Multi-Page Architecture (MPA)</b>. This guarantees zero-dependency bloat, instant LCP (Largest Contentful Paint), and maximum SEO dominance. It seamlessly connects B2C software buyers, B2B agency resellers, and digital creators into one unified hub.

<h2><img src="https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/solid/microchip.svg" width="24" /> The 500+ Functions Arsenal</h2>

This platform executes over **500+ distinct frontend functions and logical algorithms**, categorized into five major pillars:

### 🛒 1. Advanced E-Commerce & Checkout Engine
*   **Dynamic Cart System:** Real-time `localStorage` synchronization across 37 independent pages.
*   **Smart Bundle Builder:** Drag-and-drop UI that auto-calculates tiered discounts based on item count.
*   **Algorithmic Promo Engine:** Validates case-sensitive promo codes and applies percentage/fixed discounts instantly.
*   **Encrypted Checkout:** Secure form handling with multi-gateway selection (Crypto, Bank, Mobile Wallets) and Base64 proof upload simulators.
*   **Confetti & Success Overlays:** High-end post-purchase micro-interactions.

### 💼 2. B2B Reseller Enterprise & APIs
*   **Shopify-Style Live Editor:** Split-screen DOM manipulator allowing resellers to change storefront themes, colors, and logos in real-time.
*   **Margin Calculators:** Auto-calculation of wholesale vs. retail profit margins.
*   **Developer API Portal:** Interactive code snippets (cURL, Node.js) for B2B clients to integrate Nexra's auto-dispatch webhooks.
*   **Wholesale Bulk Orders:** Dynamic quote generation based on tiered quantity logic.

### 🎨 3. Creator Hub & Social Mechanics
*   **"Youtuber Bano" Upload Portal:** Simulated file chunking and progress bar UI for creators uploading PSDs/PRPROJs.
*   **3-Share Unlock Algorithm:** `localStorage` driven viral marketing loop (forces users to share a link 3 times to unlock hidden DOM elements).
*   **Live Giveaway Spin Wheel:** Verifiably random visual engine for VIP users.
*   **Community Feed:** Timeline UI with like/comment simulation and instant DOM appending.

### 🛡️ 4. Security, Gamification & User Dashboard
*   **Centralized Vault Logic:** Fetches and decrypts user-owned digital assets.
*   **Gamified XP & Achievements:** Algorithms that track user actions to unlock badges and level up profiles.
*   **Nexra Wallet & Reward Store:** Point-based exchange system (Nexra Coins -> Discount Coupons).
*   **2FA & Session Management:** UI toggles for advanced account security tracking.

### 📊 5. Analytics & Help Center Operations
*   **Live Uptime Monitor:** Visual status bars for API, Database, and Dispatch availability.
*   **Order Tracking Timeline:** Multi-step visual progress bar for physical shipments.
*   **Dynamic Knowledge Base:** Real-time character-matching search engine for FAQs and Tutorials.
*   **Feature Upvote Roadmap:** Kanban-style board with persistent local voting records.

*   **Zero Framework Dependency:** No React, No Vue, No jQuery. 100% Native DOM Manipulation.
*   **Vercel URL Masking:** Uses `vercel.json` rewrites to hide `.html` extensions and generate clean, dynamic, SEO-optimized slugs (e.g., `/p/capcut-pro`, `/u/arbab`).
*   **Automated Component Injector:** `core.js` acts as a compiler, fetching and injecting `header.html` and `footer.html` across all 37 pages to adhere to DRY (Don't Repeat Yourself) principles.

Because this platform utilizes a custom Component Fetching Engine, it cannot be opened via the file:// protocol due to standard browser CORS policies.
Clone the repository (Authorized personnel only).
Open the project in Visual Studio Code.
Install the "Live Server" extension by Ritwick Dey.
Right-click on index.html and select "Open with Live Server".
The ecosystem will boot up securely at http://127.0.0.1:5500/.
