Nexra Tech PK — Header Engine v2.0

Global Header
System

All-in-one floating island header with theme engine, slider drawer, toast notifications, accent colors, and responsive layout. Single HTML file — no external dependencies.

Dark / Light Mode

Click the moon icon (desktop) or top-right toggle (mobile). Preference saved in localStorage — persists across all pages.

Custom Accent Colors

Hold the moon button for 2 seconds to open the color picker. Or open the sidebar and pick from 9 colors. Saved globally via localStorage.

Sidebar Slider

Tap the hamburger icon anywhere to open the full right-side drawer with navigation, profile, theme, and color controls.

Toast System

Action-driven toast notifications for every user interaction. Stack-aware, auto-dismiss after 3.5s, with success/error/warning states.

Cart & Badges

Cart badge with count management. Notification badge with unread counter. Both sync with your localStorage state system.

Auth-Ready Profile

Profile button swaps icon for user photo on login. Connect window.NXR.setUser() to your auth system — Firebase or any other.
